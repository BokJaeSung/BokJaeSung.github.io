DOM (Document Object Model) 1.1 DOM이 뭔지 브라우저가 HTML 파일을 받으면 그냥 화면에 텍스트를 뿌리는 게 아니다. 브라우저는 HTML을 읽으면서 머릿속에 구조를 정리한다. 이 정리된 구조가 바로 DOM이다. HTML은 사실 그냥 메모장에 쓴 텍스트 파일이다. <h1>안녕하세요</h1> <p>저는 문단입니다</p> <button>클릭하세요</button> 브라우저는 이걸 읽으면서 이렇게 정리한다. document └── html └── body ├── h1 ├── p └── button 이 트리 구조가 DOM이다.
